# Tidal Geometry

Piers and breakwalls simplify the shoreline into clean shapes. The tide then bends those lines with reflections and foam.

This set leans into horizontal motion, keeping the frame open enough for each edge to breathe against the water.

I avoided aggressive cropping so the viewer can read distance directly from the spacing between structures.
